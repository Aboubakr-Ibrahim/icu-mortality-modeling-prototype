# ICU Mortality Modeling Prototype

A development repository for rebuilding an earlier MATLAB healthcare-modeling experiment into a transparent, reproducible, and leakage-aware portfolio project.

> **Status: under development.** This repository does not currently contain an executed or validated mortality-prediction model.

## Project background

The predecessor work explored ICU mortality modeling as a paid or voluntary biomedical research contribution. Its earlier experimental design used simulated or simplified variables and is not suitable for publication as clinical evidence.

This repository is being preserved as a development space while the methodology, data provenance, labels, validation design, and evaluation workflow are rebuilt.

## Planned work

- Define a legitimate, documented outcome label
- Use a suitable public or authorized critical-care dataset
- Specify a clinically meaningful prediction time
- Separate patients across training, validation, and test data
- Fit preprocessing only on training data
- Compare interpretable baselines before complex models
- Report discrimination, calibration, class balance, and uncertainty
- Document dataset limitations, bias, privacy, and intended use
- Add reproducible MATLAB code, tests, and reviewed outputs

## Publication rule

No performance metric, clinical claim, or model result will be published until the rebuilt pipeline has been executed and independently reviewed for methodological and implementation errors.

## Responsible use

This repository is for education, research development, and professional portfolio review only. It is not a medical device and must not be used for diagnosis, prognosis, monitoring, treatment, triage, alarms, or patient-care decisions.
